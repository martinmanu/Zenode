# @zenode/core
**The "Shared Contract" layer for the Zenode ecosystem.**

This package contains only TypeScript interfaces and type definitions. It serves as the foundation for all other Zenode packages, ensuring a type-safe, decoupled architecture.

---

## 🏗 Why `@zenode/core`?
It establishes the standard for Zenode data exchange. Whether you are building an execution engine, a visualizer, or a CLI tool, you only need to depend on this package to speak the Zenode language.

---

## 🚀 Installation
```bash
npm install @zenode/core
```

## 🛠 Key Definitions
- `ZenodeDiagramState`: The versioned snapshot of the entire diagram.
- `NodeData`, `EdgeData`: The fundamental data structures.
- `ShapeRenderer`: The contract for creating custom engine-aware shapes.
